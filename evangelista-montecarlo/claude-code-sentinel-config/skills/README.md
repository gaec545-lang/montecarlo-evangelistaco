## Política de Creación

Un skill se crea cuando:
1. Un patrón de código se repite 3+ veces
2. Un workflow manual puede automatizarse
3. Un template puede parametrizarse para reutilización

Cada skill debe tener: SKILL.md, templates/, scripts/ (si aplica), tests/.
